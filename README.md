# My-Python
🐍 Python Fundamentals Project
A Data Technician Bootcamp Project
📝 Overview
This project was completed as part of a Data Technician bootcamp, focusing on building core Python programming skills. Through a series of practical exercises, I learned how to write basic Python scripts that interact with users, perform calculations, and apply logical decision‑making. These foundational skills were applied to mini‑projects such as FizzBuzz, calculator programs, data checks, and interactive console applications.

🔧 Key Python Skills Demonstrated
🔤 Variables & Data Types

Created and used variables to store user inputs, numeric values, and program states.
Practiced type casting (e.g., converting strings to integers for calculations).
Used different data types such as integers, floats, strings, and booleans.

🖨️ Using print() and input()

Built interactive programs using input() prompts.
Displayed results, instructions, and custom messages using print().
Created user‑friendly text menus and feedback loops.

🔁 Control Flow: If Statements, For Loops, While Loops

Used if / elif / else statements to check conditions such as:

Whether numbers are even/odd
Valid menu selections
Division logic in FizzBuzz


Applied for loops for repetitive tasks like iterating through number ranges or lists.
Implemented while loops to create ongoing user interactions, such as ATM menu systems and retry prompts.


🧠 How These Skills Were Used
